✅ Методичка: как запустить проект (памятка)
1. Открыть командную строку в папке проекта
text
cd C:\Users\Павел\Desktop\Вайб Кодинг\crm-for-agents
2. Активировать виртуальное окружение
text
venv\Scripts\activate
После ввода слева появится (venv) — значит, окружение активно.

3. Запустить сервер
text
python manage.py runserver
4. Открыть в браузере
text
http://127.0.0.1:8000
✅ Когда нужно остановить сервер
В командной строке нажми Ctrl + C

✅ Когда нужно обновить код из GitHub
text
git pull origin main
✅ Когда нужно сохранить изменения в Git
text
git add .
git commit -m "что изменил"
git push origin main
✅ Если вылезла ошибка — не пугаться
Скопировать красный текст и отправить сюда — разберёмся.

✅ Короткая версия (для быстрого старта):
bash
cd crm-for-agents
venv\Scripts\activate
python manage.py runserver
Сохрани этот текст в блокнот или в файл start.txt рядом с проектом.
